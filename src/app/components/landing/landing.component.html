Instructions

1. Select a Point

Select a point on the map that you wish to create an obstacle at.

2. Description

Type in a quick description of the obstacle or event so others will know what is going on.

3. Like or Dislike

Like or dislike events to rate the obstacle or event and help other knights!

4. Delete

The event will automatically delete itself when it reaches 100 dislikes or can be manually deleted by the creator.
